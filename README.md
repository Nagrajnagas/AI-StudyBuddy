# 📚 AI StudyBuddy

AI-powered learning assistant for students.

## 🚀 Features

- 📌 AI Summary Generator
- 🧠 Quiz Generator
- 🃏 Smart Flashcards
- ❓ Doubt Solver using RAG
- 📂 PDF Upload Support

- Streamlit
- Gemini API
- Sentence Transformers
- FAISS
- PyPDF2

---

## 🧠 AI Features

- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Semantic Search
- Embedding-based Question Answering

---

## ⚙️ Installation

```bash
git clone YOUR_REPO_LINK
cd AI-StudyBuddy
```

Create virtual environment:

```bash
python -m venv venv
```

Activate venv:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run project:

```bash
streamlit run app.py
```

---
